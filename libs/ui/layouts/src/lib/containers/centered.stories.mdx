<h1 className="mb-8 text-5xl">Containers</h1>

The following components are used to horizontally center content and restrict its width.
They are commonly used throughout the entire application.

```tsx
function SomePage() {
	return (
		<div className="flex flex-col">
			<CenteredSection>
				<h1>The Heading</h1>
				<p>
					Lorem ipsum dolor sit amet, consectetur adipiscing elit. Pellentesque euismod,
					nisi eu consectetur euismod, nisl nisl consectetur nisl, eu consectetur nisl
					nisl euismod nisl.
				</p>
			</CenteredSection>
			<div>Some content that requires the entire page width.</div>
			<CenteredContainer>
				<h2>More Centered Content</h2>
			</CenteredContainer>
		</div>
	);
}
```

<h2 className="mt-16 mb-4 text-3xl">CenteredContainer</h2>

A `CenteredContainer` is a **horizontally centered** container with **restricted width (1024px)**. At small breakpoints,
it will automatically add some horizontal padding.

	<Story name="Centered Container">

<h2 className="mt-16 mb-4 text-3xl">CenteredSection</h2>

A `CenteredSection` is a wrapper for `CenteredContainer` that adds some vertical padding and uses 100% width.
You can use the `className` property to add a background color to a section.
