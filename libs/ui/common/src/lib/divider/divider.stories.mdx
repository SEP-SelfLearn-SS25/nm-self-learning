import { Canvas, Meta, Story } from "@storybook/addon-docs";
import { Divider } from "./divider.tsx";

<Meta title="Common/Divider" />

<h1 className="mb-16 text-5xl">Divider</h1>

The `Divider` can be used to create a horizontal border between sections of content.

<Canvas>
	<Story name="Divider">
		<div className="flex flex-col gap-8">
			<div>Above</div>
			<Divider />
			<div>Below</div>
		</div>
	</Story>
</Canvas>

Alternatively, you may also consider using the `divide` from Tailwind CSS, like so:

```tsx
<div className="divide-y divide-light-border">
	<div className="p-4">Above</div>
	<div className="p-4">Below</div>
</div>
```

<div className="divide-y divide-light-border">
	<div className="p-8">Above</div>
	<div className="p-8">Below</div>
</div>
