## Missing Images

When `imgUrl` is not specified, a placeholder will be rendered.

<Canvas>
	<Story name="Missing Image">
		<div className="gradient card p-8">
			<AuthorChip name="Max Mustermann" slug="max-mustermann" />
		</div>
	</Story>
</Canvas>

## Invalid URL

When `imgUrl` cannot be resolved, a placeholder will be rendered.

		</div>
	</Story>
</Canvas>
